A CLI XMas message from SUSE pack team
======================================

Run with `packxmas` in a console and enjoy.

To run locally:

1. Install dependencies, in python environment (python-poetry is required)

```
$ poetry install
```

1. Run locally

```
$ poetry run packxmas
```

This package can be also installed using [opi](https://github.com/openSUSE/opi):

```
$ sudo opi packxmas
```

And after the installation you can run the command from terminal:

```
$ packxmas
```
